September – December 2024

Summary#

I engineered a multi-class deep learning pipeline to classify 87,000+ ECG heartbeat images into five diagnostic categories to enable high-precision automated screening.

Details

  • Architectures: Transfer learning with ResNet-18 and DenseNet-121 (pretrained weights).
  • Results: Peak macro F1 score of 0.739, outperforming custom CNN baselines by ~40%.
  • Techniques: Probabilistic noise-based augmentation, inverse-square-root weighted oversampling, robust data pipeline to mitigate class imbalance and distribution shifts.
